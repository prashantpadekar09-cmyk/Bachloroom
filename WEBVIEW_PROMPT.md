# Webview Prompt

Use this prompt when you want to launch the local app in a browser or embedded webview for this project.

## Prompt

Open the local web app for `Bachloroom v1` in webview mode.

Requirements:
- Start the local server from the project root with `npm run dev`
- Wait until the server is available on `http://localhost:3000`
- Open `http://localhost:3000`
- If the server is already running, reuse it instead of starting a second copy
- If launch fails, check the server logs and report the error clearly

## Local URL

`http://localhost:3000`
